Hướng dẫn cài đặt ứng dụng
1. Cài đặt JDK theo link: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
2. Cài đặt MySQL Server: https://downloads.mysql.com/archives/community/
3. Cài đặt công cụ quản lý database Workbench hoặc SQLyog
	Workbench: https://dev.mysql.com/downloads/workbench/
	SQLyog: https://sqlyog.en.softonic.com
4. Restore database: trên công cụ quản lý database chạy file script db_milktea.sql hoặc restore
5. Chạy ứng dụng MilkTeaManagement.jar trong thư mục MilkTeaManagement/dist/
